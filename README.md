# Gojacdoors E-commerce Storefront

Welcome to the Gojacdoors E-commerce Storefront, the premier online destination for premium doors. This repository contains the source code for a robust and modern digital storefront tailored for an online door retailer. The project is designed with a focus on delivering a seamless, intuitive, and visually engaging shopping experience for all customers.

## Overview

The Gojacdoors platform provides a comprehensive suite of e-commerce features built on a reliable technology stack. It ensures that users can effortlessly browse collections, search for specific styles, read detailed product information, and proceed through checkout with minimal friction.

## Key Features

- **Intuitive Storefront Navigation:** Categorized browsing allowing users to jump directly to specific styles or types of doors.
- **Detailed Product Showcase:** Rich product pages featuring high-quality imagery, comprehensive product descriptions, and integrated user reviews.
- **Shopping Cart & Checkout:** Streamlined cart management for a smooth purchase flow.
- **User Accounts:** Secure registration and login. Customers can manage their profiles, save delivery addresses, and view their comprehensive order history.
- **Responsive & Mobile-First Design:** The UI is crafted to look and function perfectly across all devices ranging from desktops to smartphones.
- **Theme Toggling:** Integrated dark and light mode switch for user preference and enhanced accessibility.
- **Customer Support Pages:** Ready-to-read pages for FAQs, Delivery Information, Contact Us, Privacy Policies, and Terms & Conditions.

## Technologies Used

The project is built emphasizing a clear separation of concerns, high performance, and rapid UI development:

- **Frontend Styling:** [Tailwind CSS](https://tailwindcss.com/) for a utility-first, highly customizable, and responsive design system.
- **Iconography:** [Heroicons](https://heroicons.com/) (Inline SVGs) to reduce external requests and maintain a clean, modern aesthetic.
- **Interactivity:** Vanilla JavaScript for dynamic components like carousels, mega menus, tab switching, and floating UI actions.
- **Backend/Templating:** PHP handles dynamic content templating, page routing, and form processing.

## Project Structure

This project follows a straightforward architecture:

- `index.php` - The primary landing page featuring heroes, new arrivals, categories, and testimonials.
- Account-related pages (e.g., `login.php`, `signup.php`, `manage-addresses.php`, `orders.php`)
- Information/Static pages (e.g., `about-us.php`, `faq.php`, `contact.php`, `privacy-policy.php`)
- Component definitions (like CSS and shared JS utilities) residing in `theme.css` and `components.js`.
